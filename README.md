Cómo instalar dependencias:
    1. Crear un entorno virtual usando: "python -m venv .venv" (o "python3 -m venv .venv").
        - Preferible usar .venv como nombre para que lo reconozcan los IDEs
    2. Activarlo usando:
        - ".venv\Scripts\activate.bat" (Windows, cmd)
        - ".venv\Scripts\activate.ps1" (Windows, Powershell)
        - ".venv/bin/activate" (Mac/Linux)
    3. Instalar ahora sí las dependencias guardadas en requirements.txt usando "pip install -r requirements.txt".

Cómo ejecutarlo:
    1. Teniendo todo lo anterior, abrir el archivo "main.ipynb" desde VScode o desde Jupyter Lab (escribiendo en consola: "jupyter lab").
    2. Ejecutar la primera celda de código, la cual agrega la carpeta "src" al "path".
    3. Ejecutar las celdas de los ejercicios.
